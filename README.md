# Sign Aura

A real-time sign language recognition app that supports English, Telugu, and Tamil alphabets using hand gesture recognition.

## Features
- Real-time hand gesture recognition
- Support for multiple languages (English, Telugu, Tamil)
- Video call functionality
- Simple and intuitive interface

## Setup
1. Clone the repository
2. Open `app.html` in a modern web browser
3. Allow camera access when prompted
4. Start making hand gestures to see the recognized letters

## Technologies Used
- HTML5, CSS3, JavaScript
- MediaPipe for hand tracking
- PeerJS for WebRTC video calls
